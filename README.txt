Basic Arithmetic - Type in an expression using '+','-','*','/',','()'

Boolean Logic - use 'true'/'false' (or negation)

Comparison Operators = use '=='|'!='|'>'|'<'|'>='|'<=' 

Strings - Type in string, concatenate and equality/inequality

Variables - declare, and redeclare using the 'Var' keyword
            use the variable in operations
            print variable simply by typing its name eg 'num1'

Conditional Statements - IF condition DO expression
                         WHILE condition DO expression -> then show value by typing it in the command line

User Input - INPUT mystring Variable

Lists -    {}  = create empty list
           {1,2,3} = create bigger list
           {1,2,3} + 4 = add '4' to list
           {1,2,3} - 0 = remove the 1st value
           {1,2,3}== 2  = index value
